## 1. 제품 개요

### 한 줄 요약
뉴스 기사 URL 하나를 붙여넣으면, 보험전문 변호사 SNS용 인스타그램 카드뉴스 HTML을 자동으로 만들어주는 Claude Code 슬래시 커맨드.

### 해결하는 문제
보험전문 변호사가 SNS 카드뉴스를 만들려면 지금은:
1. 기사를 직접 읽고 요약
2. 8장 구조를 머릿속으로 설계
3. 미리캔버스·캔바에서 시각화
4. 인스타 규격(1080×1080)으로 내보내기

→ 전 과정이 수동이고 반복적. URL 한 줄이면 끝나야 한다.

### 핵심 가치
- **속도**: URL 입력 → HTML 완성까지 Claude Code 한 번의 실행
- **품질 고정**: 8장 구조·톤·CTA가 항상 일정 (브랜드 일관성)
- **인스타 즉시 사용**: 1080×1080 고정, 배경 이미지 자동 매칭, 바로 스크린샷 가능

---

## 2. 사용자

### 주요 사용자
- **누구**: 보험전문 변호사 1인 (본인 전용)
- **상황**: 최신 보험·법원 관련 뉴스 기사를 보고 SNS에 올릴 콘텐츠로 가공하고 싶을 때
- **목표**: 기사 → 팔로워에게 유용한 카드뉴스 → 신뢰·인지도 축적

### 사용자 시나리오
1. 변호사가 매일경제 등 뉴스에서 보험 관련 기사를 발견한다
2. Claude Code에서 `/cardnews-maker https://...` 한 줄을 입력한다
3. `50-my-work/cardnews-maker/` 에 HTML 파일이 생성된다
4. 브라우저에서 열어 8장을 확인하고 스크린샷 또는 인쇄 → 인스타에 올린다

## 4. 사용자 흐름

### 핵심 흐름
```
/cardnews-maker <URL>
  → 기사 크롤링 (insane-search)
  → 기사 분석 → 8장 콘텐츠 생성
  → 카드별 Unsplash 키워드 선택 → 이미지 URL 조립
  → 1080×1080 HTML 렌더링 (다크 테마 + CTA 삽입)
  → 50-my-work/cardnews-maker/{파일명}.html 저장
  → "브라우저에서 열어보세요" 안내
```

### 상세 흐름
1. **URL 수신**: 사용자가 `/cardnews-maker <URL>`을 입력
2. **크롤링**: insane-search engine으로 기사 본문·제목·날짜 추출
3. **콘텐츠 분석**: Claude가 기사를 읽고 8장 각각의 제목·본문·이미지 키워드 결정
4. **HTML 생성**: Tailwind CDN + 인라인 CSS로 1080×1080 카드 8개 렌더링
5. **저장 및 안내**: 파일 저장 후 경로와 다음 액션(스크린샷 방법 등) 안내

## 5. 성공 기준

- [ ] URL 1개 입력 → 60초 이내 HTML 파일 생성
- [ ] 매일경제·조선일보·연합뉴스 등 주요 뉴스 사이트 크롤링 성공률 90% 이상
- [ ] 생성된 HTML을 브라우저에서 열면 8장이 1080×1080으로 정확히 표시됨
- [ ] 각 카드 배경 이미지가 주제와 연관된 이미지로 자동 매칭됨
- [ ] 8장 CTA에 변호사 이름·연락처가 빠짐없이 들어감

## 6. 안 만드는 것 (Out of Scope — Phase 1)

- **키워드 자동 검색** — URL을 직접 주는 방식이 더 정확하므로 제외. Phase 3에서 검토
- **이미지 AI 생성** (DALL·E·Stable Diffusion) — Unsplash 직접 URL로 충분, API 키 불필요
- **인스타 자동 업로드** — 인스타 API 승인 절차 복잡, 스크린샷 수동 업로드로 충분
- **다국어 지원** — 한국어 기사 전용
- **댓글·DM 자동 응답** — 범위 초과
